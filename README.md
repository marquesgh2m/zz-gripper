# zz-gripper

####Introdução: ####
Feita no laboratório de sistemas autÔnomos da FACIN na PUCRS.
Esta gripper foi desenvolvida no laboratório visando aplicações _pick-and-place_ em conjunto com o robo Turtlebot.
